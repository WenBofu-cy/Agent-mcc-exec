## 一、仓库定位

本目录包含 Agent-mcc-exec 全部 12 个模块的接口规格文档，每个模块独立成档，严格遵循统一的十章标准模板。

Agent-mcc-exec 是 AI Agent 的行动执行层，负责接收认知大脑（Agent-ecc-brain）下发的工具调用指令，执行具体的 API 调用、代码运行、文件操作等，并返回结构化执行结果。只负责精准执行，不参与任务决策。

## 三、接口规格文档标准结构

每个模块的接口规格文档包含以下十章：

1. **基本信息**：模块编号、名称、分区、职责、依赖/被依赖模块
2. **内部状态定义**：状态机及触发条件
3. **输入数据**：数据类型、来源模块、触发条件、优先级
4. **输出数据**：数据类型、目标模块、输出条件、优先级
5. **核心处理逻辑**：伪代码描述
6. **约束与异常处理**：异常场景及恢复条件
7. **总线契约**：总线操作、数据内容、权限
8. **安全边界**：安全规则逐条列出
9. **接口校验用例**：覆盖正常/边界/异常场景
10. **质量自检清单**
